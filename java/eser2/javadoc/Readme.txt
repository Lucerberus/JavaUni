1) cercate le proprieta del sistema foto1
2) apri le variabili d'ambiente foto2
3) nelle variabili di sistema agiungete una nuova foto3
   nome: JAVA_HOME
   valore: dove avete installato java foto4

4) aprite il path foto5
5) mettere il path di java\bin foto5
